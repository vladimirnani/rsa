**Fermatt Little theorem**

$`a^{(p-1)} \equiv 1 \mod p`$


**Euler Totient function**

$`\varphi (n) = \varphi(p_1-1) * \varphi(p_2-2)`$ 

**Proof**

$`m^ed \equiv m \mod(n)`$

$`ed \equiv 1 mod (\varphi (n))`$ or $`ed = 1 + k(\varphi (n))`$

**Euler theorem**

$`a^n \equiv 1\mod \varphi (n)`$
